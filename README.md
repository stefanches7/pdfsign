PdfSign and ImageSign
=======

Edit multiple PDFs / JPEG images at once. 

Use Cases
-------
- Personal **certificates of attendance**
- Personalized letter card to multiple persons (e.g. **christmas cards**)

Installation and usage
--------
```bash
git clone https://github.com/stefanches7/pdfsign.git
cd pdfsign
pip install -r requirements.txt
```
For **PDF** run:

```bash
python pdfsign.py -h
```
For images (**JPEG, PNG etc.**):
```bash
python imgsign.py -h
```
.
